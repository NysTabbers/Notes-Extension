# Notes Extension

A lightweight browser extension to quickly create and organize notes while browsing.

## Features

- Create notes
- Automatically save page links
- Minimal interface for easy access

## How to install

This extensions hasn't been released on any extension store.

To install this extension follow the steps below

Installations may differ between browsers. I will show the installation for Chrome and Firefox Developer Edition.

First download the source code on your device

### Chrome Install

Go to [Chrome Extension](chrome://extensions/)

From here turn on developer mode on the top right. Then press Load Unpacked and add the src folder and done. It should now show up in your extensions.


### Firefox Developer Edition Install

Go to [Firefox Debug](about:debugging#/setup)

From here press This Firefox on the left sidebar. Then press Load Temporary Add-on and find the manifest.json file add it and done. It should now show up in your extensions.


## Usage

Open the extension and you can make notes. The extension will also get the link from the site you are on when you opened the extension. This is so you can easily find sites back and you know what the site was for if you have forgotten.

## Issues

This extension has only been tested on Chrome, Firefox Developer Edition and Zen Browser. Both Chrome and Firefox Developer Edition worked.
Zen Browser didn't work. If you wanted to trie it yourself on Zen you install it the same way as Firefox
